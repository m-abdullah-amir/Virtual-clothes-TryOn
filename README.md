# NextGen Smart Virtual Try-On System

An AI-powered virtual try-on web app that lets users see how a garment
will look on them using two photos: a full-body photo of themselves and
a photo of the clothing item.

## Status
🚧 In development

## Planned Stack
- Garment isolation: rembg + SAM2
- Pose/body analysis: MediaPipe Pose
- Try-on engine: CatVTON (hosted on Hugging Face Spaces, ZeroGPU)
- Frontend: Next.js + Tailwind + shadcn/ui
- Backend hosting: Hugging Face Spaces
- Frontend hosting: Vercel
